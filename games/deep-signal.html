DEEP SIGNAL

Playable / Prototype Archive

最新版: v0.5.4 mobile sonar button

AIと相談しながら制作しているレトロ深海シューティング。STAGE 1〜8、ORBITAL SIGNAL MODE、SIGNAL CORE、STAGE SELECTを搭載。v0.5.4ではスマホ用ソナーボタンを追加しました。

PLAY CURRENT VERSION BACK TO GAMES BACK TO TOP

ゲーム内容

  • STAGE 1〜8: 海中探索、ボス戦、浮上対空戦をつなぐ本編ステージ。
  • 深海ステージ: ソナーで敵影を発見し、爆雷で水中敵を撃破します。
  • 浮上対空戦: 潜水艦が海面に浮上し、空中の敵を対空弾で迎撃します。
  • ORBITAL SIGNAL MODE: STAGE 8後に展開する宇宙エンドレスモード。
  • SIGNAL CORE: 宇宙wave 10で出現する宇宙ボス。
  • STAGE SELECT: SIGNAL CORE撃破後に解放されるステージ選択機能。
  • v0.5.3では、左下の仮想スティックで倒した方向へ移動し、右側タップ/長押しショットを維持しています。
  • v0.5.4では、右下のショットボタンの上にソナーボタンを追加し、スマホでもソナーを撃てるようにしました。

操作方法

  • Arrow Keys / WASD: 移動
  • Space: 攻撃
  • E / Shift: SONAR / RADAR / SCANNER
  • P / Esc: ポーズ
  • M: サウンドON/OFF
  • R: リスタート
  • タイトル画面で O: ORBITAL SIGNAL MODE
  • タイトル画面で S: STAGE SELECT

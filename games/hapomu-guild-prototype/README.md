# HAPOMU GUILD / はぽむギルドと給食樹の迷宮

v0.1.0 の一人称3DダンジョンRPG風プロトタイプです。

## 起動

`games/hapomu-guild-prototype/index.html` をブラウザで開きます。ビルド工程、外部ライブラリ、フレームワークは使っていません。

## v0.1.0 の内容

- タイトル画面
- 固定3人のパーティ作成画面
- 8x8固定マップの1階層
- Canvasによる疑似一人称ダンジョンビュー
- 歩いた床を記録する手描き風マップ
- 扉、看板、宝箱、罠、通常敵、強敵シンボル、出口
- ターン制戦闘
- 全滅画面
- 1階クリア画面

## 操作

PC:

- W / ↑: 前進
- A / ←: 左を向く
- D / →: 右を向く
- S / ↓: 後退
- Space / Enter: 調べる、決定。戦闘中は攻撃
- M: メモ
- R: リトライ

スマホ横持ち:

- 画面下の大きなボタンで移動、調査、メモ、戦闘コマンドを操作します。

## パーティ

- オムロウ / はぽたま守り: 前衛、防御役
- プニメディ / ぷりん手当係: 回復役、打たれ弱い
- ケチャペン / 赤だれ書記: 攻撃役、MP少なめ

## 迷宮の注意

罠「あまい床」は、かわいい見た目ですが大ダメージを受けます。近くの看板にヒントがあります。

強敵「おひるね番長」は右側通路を上下に巡回します。序盤で戦うとほぼ負けますが、右側マップで位置を見れば避けられます。

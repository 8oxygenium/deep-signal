# DEEP SIGNAL v0.3.4 バランス調整メモ

v0.3.4 は、v0.3.3 の1UP、最大残機5、補給、ボス、ステージ進行、移動制限を維持しながら、ステージ個性とテンポを改善する版。

## 海中ステージの縦方向拡張

- `sea` / `seaBoss` はワールド高さを約1700pxへ拡張する。
- `air` / `airBoss` は従来の画面構成を維持し、空中戦の見え方を崩さない。
- `getWorldHeight()` で現在ステージに応じた高さを返す。
- カメラ制限、ミニマップ、海底ライン、補給、1UP、深度表現は `getWorldHeight()` を基準にする。

## 深度表現

- 深く潜るほど画面を暗くする。
- 深度ラインを増やし、縦に潜っている感覚を強める。
- 深層では敵影をより見えにくくする。
- ソナー使用時は敵の輪郭が浮かびやすくなる既存演出を維持する。
- 真っ暗になりすぎないよう、最低限のシルエットは残す。

## 高速突撃自爆敵

- 新敵タイプ `rammer` を追加する。
- 通常時は左右に高速移動する。
- プレイヤーが約160px以内に入ると警告点滅する。
- 警告後、プレイヤー方向へ高速突進する。
- 接触すると自爆し、プレイヤーにダメージを与える。
- 回避された場合は短いクールダウン後に通常移動へ戻る。
- STAGE 1には出さず、STAGE 2以降で少しずつ増やす。

## 空中戦の意味づけ

- STAGE 7 / 8 は、潜水艦が浮上して空中の敵を迎撃しているステージ。
- 自機は飛行しない。
- 海面固定、上方向迎撃、対空砲の見た目で宇宙ステージと差別化する。
- 将来の宇宙ステージで初めて全方向自由移動を許可する。

## ステージクリア後のテンポ

- STAGE CLEAR後の自動遷移は維持する。
- Space / Enter / Z で即次ステージへ進める。
- リザルト表示は、STAGE名、SCORE、LIVES、AMMO、AUTO NEXT、PRESS SPACEを優先する。
- 敵撃破数やタイムボーナスは今後の拡張候補とする。

## 維持する仕様

- 1UPは低確率ドロップ、最大残機は5。
- 最大残機時の1UP取得はスコアボーナス。
- 補給はランダム再出現制。
- `sea` / `seaBoss` は海面より上へ出られない。
- `air` / `airBoss` は空を自由に飛ばない。

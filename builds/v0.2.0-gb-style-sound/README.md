# DEEP SIGNAL v0.2.0 - GB Style Sound Prototype

## Summary
GB風ビジュアル、低解像度レンダリング、タイトル画面、Web Audio APIによるレトロSEを追加したバージョン。

## Features
- Game Boy風4階調グリーン画面
- 400x300低解像度描画
- タイトル画面
- ソナー演出
- GB風効果音
- サウンドON/OFF
- ポーズ
- 3ステージ進行

## Controls
- Arrow / WASD: Move
- Space: Depth Charge / Start
- E / Shift: Sonar
- M: Sound ON/OFF
- P / Esc: Pause
- R: Restart

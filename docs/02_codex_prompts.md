# Codex プロンプト記録

このファイルには、DEEP SIGNAL の開発で使った主な依頼内容を記録します。全文ではなく、公開時に読みやすいように要旨として整理しています。

## v0.1.0 初期プロトタイプ

依頼内容:

- HTML/CSS/JavaScriptのみで動くブラウザゲーム
- canvas使用
- 画面サイズ 800x600
- 画面上部の自機を左右移動
- Spaceで爆雷を投下
- 画面下部の敵海底ドローンを倒す
- スコア、残機、ステージを表示
- 画像素材なし、図形描画のみ
- ドット風の見た目
- 日本語コメントを多めに入れる

## 探索型への拡張

依頼内容:

- 2400x1200の広いワールド
- 自機追従カメラ
- 自機の上下左右移動
- 敵を3種類に増加
- ミニマップ追加
- 深度ライン、海底ライン、海のグリッド追加
- 敵全滅でステージクリア

## ソナー・深度・補給の追加

依頼内容:

- EキーまたはShiftキーでソナー発動
- ソナー範囲内の敵を一定時間発見済みにする
- 深度が深いほど敵を見えにくくする
- 未発見の深海敵は影として表示
- 爆雷を残弾制にする
- 補給ポイントでAMMOとLIVESを回復
- 3ステージ構成にする

## v0.2.0 GB風整理

依頼内容:

- Game Boy風の4階調グリーン調にする
- 低解像度レンダリングにする
- Web Audio APIで効果音を生成
- ソナー演出を強化
- タイトル画面を追加
- `title / playing / stageClear / gameOver` のように状態管理を整理
- バージョン番号を持たせる
- 将来のWeb版、Mobile版、GB Demake版を意識して設定値をまとめる

# DEEP SIGNAL v0.3.2 演出強化メモ

v0.3.2 は新ステージ追加ではなく、v0.3.1 のゲーム性を維持したまま、演出・視認性・ボス登場演出を強化する版。

## 海中ステージ

- STAGE 1〜6 は、暗さ・深度・潜航艇らしさを強める。
- 深い場所ほど画面全体を暗くし、海中ノイズ、泡、粒子を重ねる。
- 背景に沈んだ残骸を入れ、探索している海域の広さを出す。
- 海面より上のロック帯は、プレイ領域外として暗く無機質に見せる。
- 自機は水上艦ではなく、潜航艇・深海調査艇のシルエットとして描く。

## 空中戦ステージ

- STAGE 7〜8 は、空を広く見せる。
- 海は画面下部に抑え、水平線、雲、空のノイズ、レーダー線を強める。
- 自機は海面迎撃艇であり、空を自由に飛ばない。
- 海面の波と補給ブイの上下動で、海上から迎撃している構図を明確にする。

## ボス演出

- STAGE 6 `ABYSS CORE` と STAGE 8 `SKY SIGNAL MOTHERSHIP` には `BOSS WARNING` を入れる。
- 警告中はGB風に点滅し、短い警告SEを鳴らす。
- 警告中と登場中は敵・ボスの動きを少し抑え、ゲームが破綻しないようにする。
- `ABYSS CORE` は深海の暗闇から浮かび上がるように表示する。
- `SKY SIGNAL MOTHERSHIP` は画面上部から降下して登場する。
- 弱点やハッチが有効なタイミングは、明るい枠やバーで視覚的に分かるようにする。

## 手応え

- 被弾時に短い画面揺れを入れる。
- 敵撃破時に小さな四角パーティクルを出す。
- ボス撃破時は通常敵より大きい爆発と画面揺れを出す。
- 補給時は `DEPTH CHARGE RESTORED` / `AA SHELL RESTORED` を目立たせる。
- `STAGE CLEAR` はステージ名と `CLEAR` を中央に表示する。

## 維持する制約

- `sea` / `seaBoss` は潜航艇として海面より上へ出られない。
- `air` / `airBoss` は海面迎撃艇として空を自由に飛べない。
- 将来の `space` タイプだけが全方向自由移動になる想定を維持する。

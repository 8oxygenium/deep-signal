# DEEP SIGNAL v0.3.6 バランス調整メモ

## 目的

v0.3.6 は通しプレイ調整版です。新しい大機能は追加せず、v0.3.5 までに入れた海中探索、空中迎撃、ボス、1UP、接触ダメージ、rammer、補給、STAGE CLEARキー送りを維持したまま、難易度・テンポ・視認性・事故率を調整します。

## STAGE 1〜8 の難易度方針

- STAGE 1: かなり易しくする。敵数と弾頻度を抑え、補給を早く戻す。
- STAGE 2: ソナー、補給、接触ダメージに慣れる段階。rammerは少数に留める。
- STAGE 3: 深度と暗さを少し強めるが、rammerはまだ少数にする。
- STAGE 4: rammerと深層の暗さを本格化させる。
- STAGE 5: 深海通常ステージの締めとして手応えを出すが、暗すぎないようにする。
- STAGE 6: 初ボスとして、ソナーを使えば攻略できる難度に抑える。
- STAGE 7: 空中戦への切り替えを分かりやすくし、補給ブイを取りやすくする。
- STAGE 8: ラストボスとしてSTAGE 6より強くするが、召喚と弾幕が過密にならないようにする。

## rammer 調整方針

- STAGE 1には出さない。
- STAGE 2〜3では少数、STAGE 4〜5で本格化させる。
- 突撃前の警告時間を少し長くし、点滅と輪郭を見やすくする。
- 突撃速度を少し下げ、プレイヤーが回避できる余地を残す。
- 警告中は検出状態を少し伸ばし、深層でもソナー反応が読みやすいようにする。
- rammer接触時は自爆して消滅し、二重ダメージは発生させない。

## 接触ダメージ調整方針

- 通常敵とプレイヤーの接触判定は、見た目より小さめに維持する。
- 深層の敵はさらに少し小さめの接触判定にして、見えにくさによる事故を抑える。
- ボス本体の接触判定は弱点判定と分け、広すぎないようにする。
- 無敵時間を少し長くし、接触直後の連続被弾を防ぐ。
- ノックバック後も sea / seaBoss では海面より上へ出ず、air / airBoss では海面付近に留まる。

## 補給と1UPの調整方針

- STAGE 1〜2は補給再出現を早め、弾切れで詰みにくくする。
- STAGE 3〜5は少し探す必要がある程度にする。
- ボス戦の補給間隔を長すぎないようにし、低弾数時は再出現をさらに早める。
- 1UPは通常敵では低確率を維持する。
- rammer、UFO、魚雷艇などの強めの敵は少しだけ1UP率を上げる。
- 最大残機5と、最大残機時のスコアボーナス仕様は維持する。

## テンポと安定性

- STAGE CLEARの自動遷移時間を少し短縮する。
- Space / Enter / Z のキー送りは維持する。
- パーティクル上限を少し抑え、演出が残り続けないようにする。
- ステージ切り替え時に bombs / enemyBullets / particles / popups / oneUps が残らない仕様は維持する。

## 今後

v0.4.0 では `ORBITAL SIGNAL MODE` / 宇宙エンドレスの追加を検討します。宇宙編では初めて全方向自由移動を許可し、v0.3系で整えた接触ダメージ、補給、1UP、ボス演出の考え方を継承する想定です。

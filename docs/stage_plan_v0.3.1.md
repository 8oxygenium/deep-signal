# DEEP SIGNAL v0.3.1 ステージ調整メモ

v0.3.1 では、空中戦の見え方と補給システムを整理する。

## 空中戦の扱い

- STAGE 7 `SURFACE ALERT` と STAGE 8 `SKY SIGNAL MOTHERSHIP` は、空を自由に飛ぶステージではない。
- 自機は水上艦のまま海面付近にいて、上空の敵を迎撃する。
- 自機のY移動は海面付近の狭い範囲に制限する。
- 左右移動は広く許可し、迎撃位置を取る遊びを残す。
- Spaceキーは上方向へ飛ぶ対空弾として扱う。

## 宇宙ステージとの差別化

- 空中戦は、水平線・海面ライン・波・迎撃レーダー線を重視する。
- 画面下部だけに海を置き、画面の大部分は空として描く。
- 空中の敵が主役に見えるよう、敵の出現エリアは画面上部から中段に寄せる。
- 将来の宇宙ステージでは全方向移動や星空表現を使えるが、空中戦では使わない。

## 海中ステージの扱い

- STAGE 1〜6 の `sea` / `seaBoss` では、自機は潜航艇・深海調査艇として扱う。
- 自機は海面境界より上へ出られない。
- 海面付近では `SURFACE LOCKED` を表示し、上昇できないことを伝える。
- 画面上部の海面より上は暗いロック帯として描き、プレイ領域外であることを示す。
- 将来の `space` タイプだけが全方向自由移動になる想定で、海中・空中戦では自由飛行にしない。

## ランダム補給

- 補給ポイントは固定配置ではなく、ステージ開始時にランダム出現する。
- 補給を取ると消え、一定時間後に別のランダム位置へ再出現する。
- 通常ステージは20〜30秒程度、ボスステージは35〜45秒程度を基準にする。
- `stage.supplyRespawn` を追加すれば、ステージ単位で再出現時間を調整できる。

## stage.type ごとの補給出現

- `sea`: ワールド端を避け、水中または中層に補給ポッドを出す。
- `seaBoss`: ボスから離れた安全寄りの場所に補給ポッドを出す。
- `air`: 海面付近に浮いた補給ブイとして出す。Y座標は海面付近に固定し、X座標だけランダムにする。
- `airBoss`: 海面付近に補給ブイを出す。ボスの真下に偏らないよう、X座標を選び直す。

## 補給効果

- 補給時はAMMOを最大まで回復する。
- 残機回復は強すぎるため、低確率のボーナス扱いにする。
- 海中では `DEPTH CHARGE RESTORED`、空中では `AA SHELL RESTORED` と表示する。

## 移動ルール

### sea / seaBoss
自機は海中の潜航艇。海面より上には出られない。

### air / airBoss
自機は海面上の迎撃艇。空中へ飛行せず、海面付近から上空の敵を撃つ。

### space
将来追加予定。ここで初めて全方向自由移動を許可する。

### land
将来追加予定。地面上を走行するタイヤ付き艦として扱う。

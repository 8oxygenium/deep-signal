# DEEP SIGNAL v0.3.0 ステージ構成メモ

DEEP SIGNAL v0.3.0 では、v0.2.0 の深海探索シューティングをベースに、海中通常ステージ、深海ボス、空中戦、空中ボスまでを1本のプロトタイプとしてつなげる。

## 8ステージ構成

| STAGE | 名前 | type | 目的 |
| --- | --- | --- | --- |
| 1 | COASTAL TEST AREA | sea | 浅瀬の練習ステージ。敵が見えやすく、ソナーと爆雷の基本を確認する。 |
| 2 | SUNKEN GRID | sea | 沈んだ基地周辺。敵数を少し増やし、移動範囲の広さを見せる。 |
| 3 | MIDNIGHT TRENCH | sea | 中層から深層が中心。敵が見えにくくなり、ソナーの重要度が上がる。 |
| 4 | GHOST CURRENT | sea | 敵弾と移動敵を増やす。移動しながら先読みで爆雷を落とす練習。 |
| 5 | BLACK SIGNAL ZONE | sea | 深海編の最終通常ステージ。暗めで敵数多め。補給を使いながら探索する。 |
| 6 | ABYSS CORE | seaBoss | 深海ボス戦。ソナーで弱点を見つけ、爆雷を弱点へ当てる。 |
| 7 | SURFACE ALERT | air | 海面付近から上空の敵を撃つ対空ステージ。Spaceは対空弾になる。 |
| 8 | SKY SIGNAL MOTHERSHIP | airBoss | 空中ボス戦。弱点ハッチが開いた時だけ対空弾でダメージが入る。 |

## ボス案

### STAGE 6: ABYSS CORE

- 大型の深海コア。
- 画面下部から中層を左右に移動する。
- ソナーで発見している間だけ弱点が見える。
- 弱点に爆雷が当たるとダメージ。
- HPが半分以下になると敵弾の発射間隔が短くなる。

### STAGE 8: SKY SIGNAL MOTHERSHIP

- 上空に現れる大型UFO。
- 一定周期で弱点ハッチが開く。
- ハッチが開いている間だけ対空弾でダメージ。
- 小型UFOやヘリを召喚する。
- 下方向に弾やレーザーを撃つ。
- 撃破後は仮エンディングへ進む。

## 空中戦の仕様

- stage.type が `air` または `airBoss` の時は、モードを `SURFACE AIR` または `BOSS` として扱う。
- 自機は水上艦のままなので、上下移動は海面付近に制限する。
- Spaceキーは爆雷ではなく、上方向に飛ぶ対空弾になる。
- EキーまたはShiftキーはソナーではなくレーダーとして表示する。
- 空中敵はヘリコプター、飛行機、小型UFOを使う。
- 空中敵と空中ボスは下方向へ弾を撃つ。

## 今後の宇宙エンドレス案

v0.4.0 以降で、仮エンディング後に軌道上の信号へ進む構想。

- 表示名案: ORBITAL SIGNAL MODE
- 形式: エンドレスシューティング
- 攻撃: 通常ショット、チャージショット、短時間シールド
- 敵: 衛星ドローン、信号弾幕、軌道機雷
- 目的: スコアアタックと長時間生存
- 見た目: GB風のまま、星空と軌道グリッドを4階調で表現する

## 今後の LAND SIGNAL MODE 案

今回は実装しない陸地タイヤモードの構想。

- 表示名案: LAND SIGNAL MODE
- 形式: タイヤ付き調査車両で陸上を走る横スクロール探索
- 操作: 左右移動、ジャンプまたは段差乗り越え、地中スキャン
- 敵: 地上ドローン、固定砲台、信号ジャマー
- 目的: 海中と空中で追った黒い信号の発信源を地上で調査する
- 実装方針: 既存の `stage.type` を拡張し、`land` と `landBoss` を追加できるようにする
